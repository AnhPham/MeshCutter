MeshCutter 1.0.0 by Anh Pham

How to use:

1. Play the Demo.
2. Read DemoCutController.cs to understand how to control MeshCutter ( It's very simple ).
3. Important: MUST to use UIRoot (of NGUI) or DemoUIRoot in this package (see Demo scene).
4. Important: MUST has MeshRenderer inside the GameObject which will be cut (see Character prefab).
